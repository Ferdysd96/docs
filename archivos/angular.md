## ANGULAR


## ACTUALIZAR NPM   

``` bash
 # Actualizar nmp
 $ npm install -g npm@latest 

 # Limpiar la cahe de npm 
 $ npm cache clean --force

 # Desactivar las auditorias para evitar fallos
 $ npm set audit false
```                                          

## DESINTALAR VERSIONES ANTIGUAS  

```bash                        
 $ npm uninstall -g angular-cli
                            
 $ npm uninstall -g @angular/cli    
```

## LIMPIAR CACHE NPM

```bash    
 $ npm cache clean --force
``` 

## INSTALAR ULTIMA VERSION

```bash

```

## ANGULAR CLI 

``` bash
 # Crear nuevo proyecto 
 $ ng new name --routing    

 # Iniciar el proyecto 
 $ ng serve 

 # Crear Servicio
 $ ng g s name          

 # Crear Pipe 
 $ ng g p name
                                                                                            
 # Crear Modulos                         
 $ ng g m name  --routing                           

 # Crear Componentes
 $ ng g c name 

 # Crear Guards
 $ ng g guard name

 ## Desplegar                    
 $ ng build --prod                                
  ```
