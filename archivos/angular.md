## ANGULAR


## ACTUALIZAR NPM   

``` bash
# Actualizar nmp
 $ npm install -g npm@latest 

 # Limpiar la cahe de npm 
  $ npm cache clean --force

 # Desactivar las auditorias para evitar fallos
 $ npm set audit false
```                                          

## DESINTALAR VERSIONES ANTIGUAS


## ANGULAR CLI 

``` bash
# Crear nuevo proyecto 
 $ ng new name --routing    

 # Iniciar el proyecto  
 $ ng serve 

# Crear Servicio
$ ng g s name          

# Crear Pipe 
 $ ng g p name
                                                                                            
# Crear Modulos                         
$ ng g m name  --routing                           

#  Crear Componentes
$ ng g c nombre 

# Crear Guards
$ ng g guard name

## Desplegar                    

$ ng build --prod                                
  ```
