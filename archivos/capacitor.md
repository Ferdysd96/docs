## CAPACITOR


## COMANDS

```bash
 # installation
 $ npm install @capacitor/core
 $ npm install @capacitor/cli --save-dev
 
 # Init                                                               
 $ npx cap init                                
  
 # Build 
 $ ng build --prod
   
 # Add Plataform
 $ npx cap add android

 # Open Plataform
 $  npx cap open android  
 ```
